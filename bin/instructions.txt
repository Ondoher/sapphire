
usage: sapphire install|page|dialog|feature|vc app name [path]

Use to create the necessary boiler plate for sapphire features

options:
     install   - creates an installation of sapphire in the current directory (not implemented)
     app       - creates a new application  (not implemented)
     page      - creates a new page feature in the specified application
     dialog    - creates a new dialog in the specified application
     feature   - creates a new feature in the specified application
     vc        - creates a new empty view and controller, can specify the path to the assets folder
     app       - the name of the application
     name      - the name of the page/dialog/feature
     path      - used for vc to set the parent path of the assets folder
